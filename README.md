## 🔬 Current Research & Technical Focus

### Analytical Modeling for Low-Power Wireless Networks *(Ongoing & Confidential)*
My current primary focus involves rigorous mathematical research in the domain of physical-layer wireless communications. As the Internet of Things (IoT) expands, devices are increasingly constrained by energy limitations and the need for short-packet, low-latency transmissions.  

* **The Challenge:** Classical wireless models assume infinite blocklengths and static channels, which fail in real-world, low-power, dynamic environments.  
* **My Approach:** I formulate stochastic and probabilistic models to evaluate system performance under dynamic channel variations and temporal channel aging. Instead of relying purely on computationally heavy numerical simulations, I focus on deriving **closed-form analytical expressions**.
* **The Impact:** By successfully modeling these complex trade-offs—balancing finite-blocklength decoding reliability against strict energy-harvesting constraints—I am developing frameworks that can mathematically optimize system throughput and reliability. This work bypasses the need for resource-heavy numerical integration, providing elegant, high-accuracy approximations for practical network deployment.

## 🛠️ Core Engineering Portfolio

### 1. Real-Time Computer Vision via 5G Edge Computing
*Focus: Python, YOLOv8, Mininet, Software-Defined Networking (SDN), Network Routing*

Deploying heavy AI models on resource-constrained end-user devices often leads to thermal throttling, battery drain, and unacceptable latency. To solve this, I architected a low-latency Edge AI framework that shifts the computational burden away from the device and onto a centralized 5G Edge server.  
* **Network Simulation:** Utilizing **Mininet** and an **Open vSwitch** SDN controller, I simulated a high-bandwidth 5G core network capable of routing high-frequency UDP video packets without dropping connections.
* **System Bypass & Security:** To deploy this in a real-world restricted environment, I bypassed local network AP isolation using physical USB tethering, applying Linux kernel `iptables` and MASQUERADE NAT rules to establish a secure, direct data pipeline.
* **AI Inference:** I deployed a highly optimized **YOLOv8** object detection model paired with OpenCV on the edge server. The system successfully ingested live HD video feeds, processed the frames, and returned rendered bounding boxes in real-time, proving the viability of Edge AI offloading for critical use cases like drowning detection.

### 2. Multi-Paradigm Medical AI Diagnostic Pipeline
*Focus: Python, Scikit-Learn, Object-Oriented Programming (OOP), Data Science*

Healthcare data is notoriously complex, blending structured vitals, temporal progression, and unstructured clinical notes. I built an Object-Oriented Machine Learning pipeline that tackles this complexity by cascading patient data through multiple AI paradigms: Rule-Based, Unsupervised, Discriminative, and Generative.  
* **Feature Engineering & Discriminative AI:** Applied Principal Component Analysis (PCA) for dimensionality reduction and utilized Gaussian Mixture Models (GMM) to discover hidden patient phenotypes. This data was fed into a Multi-Layer Perceptron (MLP) Neural Network, achieving an 80.5% baseline classification accuracy.
* **Generative NLP Triage:** Engineered a Natural Language Processing module using Count Vectorizer and Multinomial Naive Bayes to classify unstructured clinical notes, fusing text insights with physical patient vitals early in the pipeline.
* **Sequential Modeling:** To model the latent progression of disease over time, I developed and compared dynamic sequential frameworks, evaluating Maximum-Entropy Markov Models (MEMM) utilizing Logistic Regression, Hidden Markov Models (HMM), and Conditional Random Fields (CRF).

### 3. Enterprise Secure Network Topology Architecture
*Focus: Cisco Packet Tracer, Network Security, Routing Protocols*

A robust software application is only as secure as the network it runs on. To solidify my understanding of enterprise-grade networking, I designed and simulated a comprehensive, secure network topology.
* **Scalable Architecture:** Integrated multiple subnets, Virtual LANs (VLANs), and dynamic routing protocols to ensure high availability and seamless scalability across the network.
* **Deep Security:** Implemented robust security measures by configuring Access Control Lists (ACLs) and stateful firewall rules. This successfully isolated broadcast domains, restricted unauthorized traffic, and safeguarded critical server subnets.
* **Layer 2 Defense:** Configured vital Layer 2 security protocols, including Switch Port Security and strict VLAN access controls, to mitigate internal network threats and prevent unauthorized end-device manipulation.
